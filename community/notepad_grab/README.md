# notepad_grab

Beacon Object File that extracts text content from open Notepad windows.

Based on [NoteThief](https://github.com/trainr3kt/NoteThief) by trainr3kt.

## Usage

```
beacon> inline-execute notepad_grab.x64.o
```

## Output

```
[*] Starting COFF Execution...

[+] *example title - Notepad

sample notepad content

lorem ipsum

[*] COFF Finished. 
```
