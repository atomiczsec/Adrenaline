# session_view 

## Summary

Beacon Object File that enumerates Windows Terminal Services sessions, displaying session IDs, usernames, domains, connection states, and session LUIDs.

Original source: [SessionView](https://github.com/lsecqt/SessionView) by lsecqt. 

## Example Output

### Typical Output
```
[*] Starting COFF Execution...

[+] 0 (Services): Disconnected - (No user logged in)
[+] 17 (Console): Active - teeeest\polo - LUID:0x00000000:0d344e42

[*] COFF Finished.
```
